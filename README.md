# Amplio Wealth Screen Redesign – UX Prototype

This repository showcases a UX prototype for the redesigned Amplio wealth screen, focused on a cleaner, more intuitive, and user-centric experience for managing investments.

## Overview

The prototype focuses on improving user engagement by streamlining key sections:
- Removed the "UPDATE YOUR DEMAT DETAILS" prompt for a less cluttered interface.
- Renamed "Asset Overview" to "My Assets" and "Financial Goals" to "My Goals" to personalize the experience.

Explore the live prototype here:  
https://v0.app/chat/wealth-screen-redesign-usCbwttonki

## My Assets – Structured Table View

The “My Assets” section presents holdings in a clear tabular format with:
- Asset class and percentage allocation displayed on the left.
- Current value or action link on the right.
- EPF status shown as a clickable link for quick access.

This layout enhances readability and quick analysis of portfolio composition.

## My Goals

“My Goals” aligns users’ financial objectives with their assets, offering immediate progress visibility and easy navigation between assets and goals.
